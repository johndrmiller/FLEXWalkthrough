The Differentiated Instruction pop-up shows which students require the completion of either an Individualized Instruction lesson or Mastery Check. These materials can be accessed and printed by using the “Printables” button from the dashboard.
Next
The Printables page shows the teacher which students require additional resources for the day, and prepares these resources as printable materials. These printable materials include Individualized Instruction lessons, Mastery Checks, and more.
If using The Print or Project Experiences, the ePresentation icon launches the ePresentation associated with the day’s lesson. ePresentations are loaded with features and functionality that enhance teacher instruction and student learning.
The Class Snapshot page shows up-to-date data on each student, including Lexile measure, words correct per minute score, current Digital Experience lesson, current Digital Experience score, and overall program score. This snapshot is easily exportable as an Excel spreadsheet.
The On Grade Level pop-up identifies which students are reading at grade level based on their current Lexile scores. 
The Teacher Dashboard is the perfect place for the teacher to start their day, as it provides learning objectives for the class day, as well as just-in-time reporting and printables to assist the teacher in organizing daily instruction.
The Recommended Readers pop-up lets the teacher know which students’ Lexile levels are high enough that they may read sections from the day’s Print Experience selection aloud to the group.
