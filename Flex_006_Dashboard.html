If using The Print or Project Experience, the Planner tab provides the teacher a month view of the curriculum assignments for each Experience, and the ability to adjust dates and change curriculum.
Next
After selecting a particular day in the Planner, the teacher can push that day’s assignments to the following in-class day by selecting the Remove button under Edit Day. This feature comes in handy for days when students are not in class. (The day’s assignments can be restored by selecting the Restore button under Edit Day.)
After selecting a particular day in the Planner, the teacher can add Print Experience and Project Experience curricula by selecting the respective buttons under Add Experience. Dropdown menus will appear for the teacher to select assignment options for the Experience being added.
